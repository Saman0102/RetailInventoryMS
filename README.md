# Inventory & Order Management System
A robust, Object-Oriented Programming (OOP) based Python application designed to streamline product tracking and order processing for small-to-medium enterprises.

# Key Features
1. Dynamic Inventory Tracking: Automated CRUD (Create, Read, Update, Delete) operations for product management.

2. Smart Order Processing: Real-time stock validation that prevents over-ordering and automatically updates inventory levels.

3. Auto-ID Generation: Sequential ID assignment for both products and orders to ensure data integrity.

4. Data Persistence: (Planned/Implemented) Capability to save and load state using JSON/File I/O.

# Technical Stack
1. Language: Python 3 
2. Concepts: Class Methods, Encapsulation, State Management, List Comprehension.
